# 夕食カレンダー生成プロンプト（Claude Code引き継ぎ用）

## プロジェクト概要

世田谷区立保育園に通う15ヶ月の子供がいる家族（大人2人）向けの夕食カレンダーを毎月生成するプロジェクト。
生成したHTMLファイルをGitHub Pagesで公開している。

**公開URL:** https://ryuhoota-stack.github.io/dinner-calendar  
**リポジトリ:** https://github.com/ryuhoota-stack/dinner-calendar  
**ローカルパス:** ~/my-dev/dinner-calendar/

---

## ⚡ 重要：UIとデータは分離済み（毎月の作業は最小限）

`index.html` は **UIエンジン（CSS・JS）だけ** を持つ。**毎月これを作り直さないこと。**
月ごとの献立は `data/YYYY-MM.js` に分かれていて、`registerMonth({...})` で登録する。

**新しい月を追加する手順（これだけ）:**
1. `data/YYYY-MM.js` を新規作成（既存月ファイルをコピーして中身を差し替えるのが速い）
2. `index.html` の最下部にある月データの `<script>` リストに1行追加
   ```html
   <script src="data/2026-07.js"></script>
   ```
3. **古い月の `<script>` は残してOK**（自動で「最新2ヶ月」だけ表示される）

**曜日・日付の整合性について:**
- 曜日の並びと日数は `new Date(year, month-1, 1)` から **動的に計算** される。
  データファイルの `year` と `month` さえ正しければ、月ごとの曜日ズレ・日数違い（30/31/28日）は構造的に起きない。
- ただし **祝日（`holidays`）は手動指定** が必要。その月の祝日を必ず確認して `{日: "ラベル"}` で入れること。
- 献立データの日付キー（`D` / `SH` / `SHOP_ITEMS`）が、その月に存在する日付かどうかは必ずチェックする。

**表示仕様（UI側で実装済み・データ追加だけで効く）:**
- 画面上部のセグメントコントロールで「先月／最新月」を切り替え（最新2ヶ月のみ表示）
- 詳細パネルの日付左右に矢印（‹ ›）。前後の日に遷移でき、月をまたぐと自動で表示月も切り替わる
- カラーはApple風。ライトはカード＋淡い影でコントラスト確保、ダークは真っ黒を避けた elevated グレー

## 毎月の作業手順（献立を考える部分）

### 1. 保育園の献立表を取得
- 世田谷区子ども・若者部保育課が毎月発行する献立表PDFを用意する
- 各日付のメイン料理・副菜を抽出する

### 2. 夕食献立を生成（以下の条件を守ること）

**絶対条件**
- 保育園の昼ご飯と夕食のメイン食材・料理ジャンルが被らないこと
  - 例：保育園がカレーの日 → 夕食はカレー以外にする
  - 例：保育園が魚料理の日 → 夕食は肉料理にする
- 日曜日は「お休み」表示（夕食献立なし）
- 土曜日は「お休み」表示（夕食献立なし）

**献立の方針**
- 大人2人分
- 買い物回数を減らせる食材の使い回しを意識
- 作り置きできるメニューを週に2〜3回入れる
- 時短（15分以内）メニューも週に2〜3回入れる
- コスパの良い食材（もやし・豆腐・卵・キャベツ等）を積極活用

**買い物日のルール**
- 日曜日（パパが一緒に来られる）: 肉・米・野菜など重いもの・量が多いものをまとめ買い
- 水曜日: 卵・豆腐など軽いものの補充のみ
- 各日の買い物リストに食材名と数量（g・個数）を明記
- 重いものには「重め」フラグを立てる

## 生成するHTMLの仕様

### ファイル構成
- 単一の `index.html` ファイル（CSS・JSすべて同梱）
- 外部依存: Tabler Icons CDN のみ
  ```html
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@tabler/icons-webfont@2.44.0/tabler-icons.min.css">
  ```

### UIコンポーネント

**カレンダーグリッド**
- 7列グリッド（日〜土）
- 各セルに日付・メイン料理名・バッジ（作り置き/時短）を表示
- 買い物日は緑背景（#E1F5EE）
- 日曜は赤系文字色（#993C1D）
- 土曜は青系文字色（#185FA5）

**詳細パネル（日付タップで表示）**
- 夕食メイン + 「レシピを見る」テキストリンク（横並び）
- tipsボックス（#E1F5EE背景、💡アイコン）
- 副菜・汁物
- 保育園の昼ご飯（被り防止表示）
- 買い物日のみ「夕食」「買い物リスト」タブ切り替え

**日曜日の詳細パネル**
- 週ごとに変わる労いメッセージ（5パターン）
  1. 「今週もお疲れさまでした！日曜日はゆっくり休みましょう。」
  2. 「1週間頑張りましたね。今日はのんびり過ごしてください。」
  3. 「お疲れさまでした。日曜日は家族でゆっくりとした時間を。」
  4. 「今週もお疲れさまでした。今日くらいはゆっくり休んでください。」
  5. 「1ヶ月お疲れさまでした！しっかり休んで英気を養いましょう。」
- メニュー選択ドロップダウン → レシピ表示（料理する場合用）
- 買い物日（日曜）は「レシピ」「買い物リスト」タブ切り替え

**レシピポップアップ（ボトムシート形式）**
- 「レシピを見る」タップでiOSライクなボトムシートモーダル表示
- 作り方（ステップ）
- ポイント（2人分の目安量など）
- クックパッドへのリンク

**買い物リスト**
- チェックボックス付き（✅でき次第チェック）
- 食材名・数量・「重め」バッジ
- 進捗カウンター（例：3/12 完了）
- 常備品セクション（米・卵・豆腐・みそ・しょうゆ・みりん・ごま油・片栗粉・冷凍うどん・キムチ・缶詰類）
- 日曜買い物日には「パパと一緒に行く日」メッセージ

### アクセシビリティ（WCAG 2.1準拠）
- タップターゲット最小44×44px
- 本文フォント最小14px、主要テキスト16px
- role="button" / tabindex="0" / aria-label をセルに付与
- ダークモード対応（prefers-color-scheme: dark）

### カラーパレット
```css
/* メイン */
--teal: #1D9E75;        /* 買い物日・チェックボックス */
--teal-light: #E1F5EE;  /* tips背景・買い物日セル背景 */
--teal-dark: #085041;   /* tipsテキスト */

/* 曜日 */
--sun: #993C1D;   /* 日曜 */

/* バッジ */

```

## データ構造テンプレート

月ごとのファイル `data/YYYY-MM.js` は丸ごと `registerMonth({...})` を1回呼ぶだけ。

```javascript
registerMonth({
  key: "2026-07",        // "YYYY-MM"
  year: 2026,
  month: 7,              // 1-12（曜日・日数はここから自動計算される）
  holidays: { 21: "海の日" }, // {日: "ラベル"} 祝日は手動で入れる。なければ {}

  // 夕食データ（保育園の昼ご飯 n と被らせないこと）
  D: {
    1: {
      m: "料理名",           // メイン料理（カレンダーのセルに出る）
      s: "副菜・ご飯",       // 副菜・汁物
      n: "保育園の昼ご飯",   // 被り防止の表示用
      t: "コツや作り置きメモ", // tips（💡ボックス）
      b: "p",               // "p"=作り置き / "q"=時短
    },
    // ...日曜・土曜・祝日は entry を入れない（自動で「お休み」表示）
  },

  // 買い物日（日曜=重いものまとめ買い / 水曜=軽い補充）
  SH: [5, 8, 12, 15, 19, 22, 26, 29],

  // 買い物リスト（日付ごと）
  SHOP_ITEMS: {
    5: { papa: true, items: [   // papa:true=日曜（パパと一緒メッセージ）
      { n: "食材名", q: "数量", h: true },  // h:true は「重め」バッジ
      // ...
    ]},
    // 水曜の補充日は papa:false
  },

  // レシピ（料理名 → 作り方）。SUN_MENUS 未指定ならこのキー全件が日曜の選択肢になる
  RC: {
    "料理名": { steps: ["手順1", "手順2", "手順3"], pt: "2人分：材料と分量のポイント" },
    // ...
  },
});
```

> ⚠️ `index.html`（UIエンジン）は触らない。新しい月の `<script src>` を1行足すだけ。

## デプロイ手順

```bash
cd ~/my-dev/dinner-calendar

# data/YYYY-MM.js を追加し、index.htmlに<script>行を1行足したあと
git add data/ index.html
git commit -m "○月献立を追加"
git push
```

pushして約1〜2分でGitHub Pagesに自動反映される。

## 参照した過去の献立（被り防止の実績）

| 日付 | 保育園 | 夕食（修正例） |
|---|---|---|
| カレーの日 | カレーライス | 豚の生姜焼き or 麻婆豆腐に変更 |
| ハンバーグの日 | ハンバーグ | 魚料理 or 炒め物に変更 |

---

## 注意事項

- 保育園は土日祝日休み（献立なし）
- 夕食も土日は「お休み」表示（日曜は労いメッセージ表示）
- 買い物リストは**その日の分だけ**表示（週まとめ表示はしない）
- チェック状態はページリロードで消える（ブラウザメモリのみ）
